Wie diese Slides entstehen
Diese Slides werden mithilfe von reveal.js und GitHub-Pages dargestellt.
reveal.js ist eine Bibliothek, die es ermöglicht Slides wie diese in HTML oder Markdown zu schreiben.
Dazu wurde ein neues Repository in GitHub angelegt. Die Pages-Funktion wird aktiviert, indem ein Branch und Ordner in den Einstellungen ausgewählt werden.
