1. API Design mit OpenAPI


Die OpenAPI (vormals Swagger) Spezifikation ist ein Standard zur Beschreibung von REST-APIs.
OpenAPI Initiative (OAI) als Teil der Linux Foundation ist für die Weiterentwicklung der Spezifikation zuständig.
Mit Hilfe des openapi-generators können aus einer OpenAPI-Spezifikation Code-Snippets generiert werden.
In Projekten, die mit Maven gebaut werden, kann der openapi-generator als Plugin eingebunden werden.
Dieses Plugin heißt openapi-generator-maven-plugin.
Die api.yml enthält die OpenAPI-Spezifikation.
Diese besteht aus Metadaten, wie z.B. dem Titel der API, und der Beschreibung der Endpunkte.
Um die Endpunkte beschreiben zu können, werden zusätzlich die Objekte beschrieben, mit denen die API arbeitet. 

           paths:
              /todos:
                get:
                  summary: List all todos
                  operationId: listTodos
                  tags:
                    - todos
                  responses:
                    '200':
                      description: A list of todos.
                      content:
                        application/json:
                          schema:
                            type: array
                            items:
                              $ref: '#/components/schemas/Todo'
        

            components:
              schemas:
                Todo:
                  type: object
                  required:
                    - id
                    - content
                  properties:
                    id:
                      type: integer
                      format: int64
                    content:
                      type: string
                    tag:
                      type: string
