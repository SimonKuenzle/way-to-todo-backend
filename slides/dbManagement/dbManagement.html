2. DB-Management


Liquibase

Was ist Liquibase?

Liquibase ist ein Tool zur Versionsverwaltung von Datenbanken. Es ermöglicht das Erstellen, Verwalten und Anwenden von Änderungen an Datenbanken.

Changesets

Änderungen an der Datenbank werden in sogenannten Changesets definiert. Ein Changeset ist eine XML-Datei, die eine oder mehrere Änderungen an der Datenbank beschreibt. Ein Changeset kann beispielsweise eine neue Tabelle erstellen, eine Spalte hinzufügen oder einen Index erstellen.

Über einen Context kann festgelegt werden, in welcher Umgebung ein Changeset angewendet werden soll.
Dies wird genutzt, um Demo bzw. Testdaten nur dann zu erstellen, wenn die Anwendung im dev-Profil gestartet wird.
Über den Lock-Mechanismus erkennt Liquibase, ob die Datenbank bereits aktualisiert wurde.
Dafür werden in der Datenbank Tabellen angelegt, die den Status der Datenbank beinhalten.
