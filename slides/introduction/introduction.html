0. Vorbereitung


Die Entstehung dieser Slides
Diese Slides werden mithilfe von reveal.js und GitHub-Pages dargestellt.
              graph TD
                A[reveal.js] -->|Slides in HTML oder Markdown| G[Github Repository]
                G -->|Pages-Funktion| B[Browser]
                B -->|Slides werden dargestellt| C[Benutzer]
Die Pages-Funktion muss aktiviert werden.
Dazu wird unter Einstellungen > Pages Branch und Ordner ausgewählt, die genutzt werden sollen.
Für die verbesserte Übersicht der Slides im Quellcode wird ein Plugin[reveal_external] verwendet, das die Auslagerung von HTML in Dateien ermöglicht.
Auch für die Darstellung von Mermaid-Diagrammen wie hier bereits verwendet wurde und das Einbinden von Markdown werden Plugins verwendet.
